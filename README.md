# MPI 并行计算作业与课设

中国地质大学（武汉）并行计算课程作业及课程设计。

## 目录结构

| 目录 | 说明 |
|------|------|
| `MPI_hello/` | MPI 入门：并行归并排序（二分归并） |
| `homework2/` | 同 MPI_hello，独立作业副本 |
| `homework3/` | MPI 矩阵乘法：3×3 矩阵，9 进程各算一个元素 |
| `homework4/` | 基于 MPI_Send/Recv 实现 MPI_Gather（HMPI 风格） |
| `homework5/` | 基于 MPI_Send/Recv 实现 MPI_Allgather（环形算法 + HMPI） |
