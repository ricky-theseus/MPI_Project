\subsection{结果分析}

从实验结果可以得出以下关键结论：

\begin{enumerate}
    \item \textbf{dEA 最优：}主从式 dEA（均值 51606，标准差 112）在求解质量上显著优于 HdEA 和 HdEA-MP，且结果高度稳定。dEA 的优势来自于全局共享种群——200 个个体在同一池中竞争、交配，提供了最充分的基因多样性，避免了局部最优的早熟陷阱。

    \item \textbf{HdEA 次之：}分层 HdEA（均值 56596，标准差 746）的求解质量明显低于 dEA。根本原因在于子种群规模过小（每岛仅 50 个个体）。在 442 城市的搜索空间中，50 个个体的种群难以维持足够的多样性，导致各岛屿较快收敛到不同的局部最优。虽然每 20 代进行一次精英迁移，但迁移仅交换最优个体，对恢复种群多样性的帮助有限。

    \item \textbf{HdEA-MP 最差但方差最大：}HdEA-MP（均值 76036，标准差 3081）的求解质量最差，但运行时间最短（约 13 秒）。值得注意的是其极高的标准差（3081），说明环形迁移机制在某些运行中确实引入了多样性，但由于岛屿规模太小，迁移的优质基因无法在种群中有效传播。此外，跑出最优结果的运行也仅达到 71605，仍远逊于 dEA 的最差结果。

    \item \textbf{dEA 与 HdEA-MP 的精度--时间权衡：}dEA 每代需要 2 次全局通信（Bcast + Gatherv），通信开销使每代耗时约 0.22 毫秒；HdEA-MP 大部分时间本地进化、仅每 20 代进行一次小规模环形通信，每代耗时仅约 0.07 毫秒。虽然 dEA 耗时大约是 HdEA-MP 的 3 倍，但 dEA 的解质量提升（降低约 32\% 的路径距离）远超额外通信的成本，在精度优先的场景下 dEA 是更优选择。
\end{enumerate}
